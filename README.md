# baterPonto
Site responsivo para controle de tarefas, um calendário dinâmico. Tecnologias: PHP, JavaScript, MySQL, SQL, HTML, CSS e Bootstrap.

Features:
- Front-end base
- Filtros por data
- Customização de layout
- Adiconar tarefas
